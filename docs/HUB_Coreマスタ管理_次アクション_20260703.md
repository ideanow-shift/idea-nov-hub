### 実装済みガード

HUB master-admin画面:

- 役職選択肢は `FORMAL_EMPLOYEE_POSITION_LABELS` に含まれるものだけ表示します。
- `会長夫人 / 創業者夫人 / 夫人` は表示対象から除外します。
- `レセプション / スタイリスト / アシスタント / 本部スタッフ` など職種系ラベルは役職選択肢に出さない構造です。

nov-hub-api:

- `assertAllowedEmployeePosition()` で正式役職リスト外の保存を拒否します。
- `会長夫人 / 創業者夫人 / 夫人` は役職として保存不可です。
- `is_active=false` の役職も保存不可です。
